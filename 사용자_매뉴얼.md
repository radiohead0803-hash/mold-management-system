## 1. 시스템 개요

### 1.1 시스템 목적
CAMS 금형관리 시스템은 금형의 전체 생명주기를 디지털화하여 관리하는 통합 솔루션입니다.

### 1.2 주요 기능
- **금형 정보 관리**: 금형 기본정보, 사양, 이미지 관리
- **점검 관리**: 일상점검, 정기점검 기록 및 이력 관리
- **수리 관리**: 수리 요청, 진행, 완료 이력 관리
- **타수 관리**: 누적 타수 기록 및 알림
- **QR 기반 접근**: 모바일 QR 스캔을 통한 빠른 접근
- **GPS 위치 추적**: 실시간 금형 위치 및 작업 위치 기록
- **실시간 알림**: 점검 예정, 수리 요청 등 실시간 알림

### 1.3 사용자 구분
- **관리자**: 본사 관리자 (보전/품질/개발 담당자)
- **협력사**: 금형 제작 및 수리 협력사
- **작업자**: 현장 작업자 (QR 스캔 접근)

## 2. 관리자 시스템

### 2.1 로그인

#### 접근 방법
- URL: `http://[서버주소]/admin/login`

- 부서별 권한 설정 가능

---

### 2.2 관리자 대시보드

#### 페이지 개요
관리자 대시보드는 금형 관리 현황을 한눈에 파악할 수 있는 메인 화면입니다.

#### 주요 섹션

##### 2.2.1 KPI 카드
**관리 항목:**
- **총 금형 수**: 시스템에 등록된 전체 금형 개수
- **점검률**: 금형별 점검 완료율 (목표: 90% 이상)
- **예정 점검**: 향후 7일 이내 예정된 점검 건수
- **수리 진행**: 현재 수리 진행 중인 금형 개수

**관리 기준:**
- 점검률 90% 미만 시 경고
- 예정 점검 3일 이내 건은 우선 처리
- 수리 진행 건은 일일 모니터링 필요

##### 2.2.2 금형 위치 맵
**관리 항목:**
- 공장별/구역별 금형 배치 현황
- 금형 상태별 색상 구분
  - 🟢 생산중 (녹색)
  - 🟡 정비중 (노란색)
  - 🔴 수리중 (빨간색)
  - ⚪ 대기중 (회색)

**관리 기준:**
- 금형 위치는 GPS 좌표 기반 자동 업데이트
- 지정 위치 이탈 시 알림 발생

##### 2.2.3 최근 활동
**관리 항목:**
- 일상점검 기록
- 수리 요청
- 타수 기록
- 상태 변경 이력

**관리 기준:**
- 실시간 업데이트
- 긴급 건은 상단 고정 표시

#### 빠른 작업 메뉴
1. **새 금형 등록**: 금형 정보 신규 등록
2. **금형수리 이력관리**: 협력사 수리 요청 및 진행 현황
3. **협력사 습합세척 이력**: 협력사 작업 기록 조회
4. **점검 일정 관리**: 예정된 점검 일정 확인
5. **QR 점검**: QR 스캔 점검 상태 확인
6. **일상점검 이력**: 작업자 점검 기록 조회
7. **알림 설정**: 시스템 알림 설정

### 2.3 금형 관리

#### 2.3.1 금형 목록 조회

**페이지 경로:** `/admin/molds`

**관리 항목:**
- 금형 ID
- 품번
- 품명
- 대상차종
- 제작처
- 상태
- 누적타수
- 최근점검일
- 위치

**검색 및 필터:**
- 텍스트 검색: 품번, 품명, 차종
- 상태 필터: 생산중, 수리중, 점검예정, 보관중
- 제작처 필터

**관리 기준:**
- 생산중 금형: 일일 점검 필수
- 수리중 금형: 수리 진행 상황 모니터링
- 점검예정 금형: 점검 일정 확인 및 준비

#### 2.3.2 금형 등록

**페이지 경로:** `/admin/molds/register`

**필수 입력 항목:**

**기본 정보**
- 금형 ID (고유번호)
- 품번
- 품명
- 대상차종
- 제작처
- 제작일자
- 금형 상태

**사양 정보**
- 재질
- 중량
- 치수 (가로×세로×높이)
- 캐비티 수

**사출조건** (선택)
- 사출온도
- 금형온도
- 사출압력
- 사출속도
- 보압
- 보압시간
- 냉각시간
- 사이클타임
- 스크류속도
- 배압

**관리 정보**
- 담당자
- 위치
- 최대 타수
- 점검 주기

**이미지**
- 금형 이미지
- 제품 이미지

**관리 기준:**
- 필수 항목 미입력 시 등록 불가
- 금형 ID 중복 체크 필수
- 이미지는 5MB 이하, JPG/PNG 형식

#### 2.3.3 금형 수정

**페이지 경로:** `/admin/molds/:id/edit`

**수정 가능 항목:**
- 기본 정보 (금형 ID 제외)
- 사양 정보
- 사출조건
- 관리 정보
- 이미지

**관리 기준:**
- 수정 이력 자동 기록
- 중요 정보 변경 시 승인 필요

### 2.4 금형수리 이력관리

**페이지 경로:** `/admin/molds/repair`

#### 관리 항목

**수리 요청 정보**
- 요청번호
- 금형 ID
- 요청일시
- 요청자
- 긴급도 (긴급/높음/보통/낮음)
- 문제 유형
- 상세 설명
- 첨부 이미지

**수리 진행 정보**
- 수리 상태 (요청됨/진행중/완료)
- 담당 협력사
- 수리 시작일
- 수리 완료일
- 수리 내용
- 교체 부품
- 수리 비용

**관리 기준:**
- 긴급 요청: 4시간 이내 대응
- 높음 요청: 24시간 이내 대응
- 보통 요청: 3일 이내 대응
- 완료 후 품질 확인 필수

#### 수리 상태별 관리

**요청됨**
- 협력사 배정
- 예상 수리 기간 입력

**진행중**
- 진행 상황 업데이트
- 추가 문제 발견 시 보고

**완료**
- 수리 결과 확인
- 품질 검사
- 금형 상태 업데이트

### 2.5 점검 관리

#### 2.5.1 일상점검 이력

**페이지 경로:** `/admin/daily-inspection-history`

**관리 항목:**
- 점검일시
- 금형 ID
- 점검자
- 점검 항목별 결과
- GPS 위치
- 첨부 이미지
- 특이사항

**점검 항목 카테고리:**
1. **기본정보**: 점검일자, 점검자, 점검위치
2. **금형 외관**: 외관 상태, 방청유 도포, 변호판/QR 상태
3. **금형 기능부**: 게이트, 냉각라인, 에젝터, 슬라이드/리프터
4. **생산 관련**: 금형온도, 운행상태, 이형제
5. **보관관리**: 보관위치, 보관상태
6. **이상조치**: 이상내용, 조치내용

**관리 기준:**
- 생산중 금형: 일일 1회 이상 점검
- 불량 항목 발견 시 즉시 조치
- 이상 발견 시 자동 알림 발송

#### 2.5.2 정기점검 관리

**점검 주기:**
- **1차 정기점검**: 100,000 타수마다
  - 금형 외관, 냉각라인, 게이트 확인
- **2차 정기점검**: 500,000 타수마다
  - 캐비티/코어, 마모, 슬라이드, 에젝터 점검
- **3차 정기점검**: 1,000,000 타수마다
  - 전체 분해 점검, 주요 부품 교체 판단

**점검 항목:**
1. **기본정보**: 금형번호, 누적 쇼트수
2. **외관/구조**: 금형 외관, 캐비티/코어, 파팅라인, 게이트부, 런너부/핀
3. **냉각/유압**: 냉각라인, 유압라인
4. **작동부**: 슬라이드/리프터, 에젝터 핀, 가이드핀/부싱
5. **부품상태**: 볼트/패킹류, 센서류
6. **금형성능**: 제품 불량률, 냉각 효율, 금형수축률
7. **조치사항**: 점검결과 종합, 수리내역, 차기점검 예정

**관리 기준:**
- 점검 예정일 7일 전 알림
- 점검 결과 "수리요" 시 즉시 수리 계획 수립
- 점검 완료 후 차기 점검일 자동 설정

#### 2.5.3 점검 일정 관리

**페이지 경로:** `/admin/molds/schedule-management`

**관리 항목:**
- 금형별 점검 일정
- 점검 유형 (일상/정기)
- 예정일
- 담당자
- 진행 상태

**관리 기준:**
- 주간 점검 계획 수립
- 담당자 배정
- 점검 완료 확인

**페이지 경로:** `/worker/mold/:moldId/daily-check`

#### 점검 프로세스

- 점검일자 (자동)
- 점검자 (자동)

**2. 점검 항목 체크**

**금형 외관**
- 금형 외관 점검 (양호/불량)
- 방청유 도포상태
- 변호판/QR 상태
- 사진 촬영

**금형 기능부**
- 게이트 상태
- 냉각라인 상태
- 에젝터 작동
- 슬라이드/리프터 작동
- 사진 촬영

**생산 관련**
- 금형온도 입력 (℃)
- 금형운행상태
- 이형제 잔량/상태

**보관관리**
- 보관위치 (GPS 자동검증)
- 금형보관상태
- 사진 촬영

**이상조치**
- 이상내용 (텍스트 + 사진)
- 조치내용 (텍스트)

**3. 타수 기록**
- 이전 타수 (자동)
- 현재 타수 (입력)
- 증가량 (자동 계산)

**4. 제출**
- 필수 항목 확인
- GPS 위치 확인
- 서버 전송

**관리 기준:**
- 필수 항목 미입력 시 제출 불가
- 불량 항목 발견 시 사진 필수
- 이상 발견 시 자동 알림 발송
- GPS 위치 오차 50m 이내

### 4.4 수리 요청

**페이지 경로:** `/worker/mold/:moldId/repair-request`

#### 요청 프로세스

**1. 기본 정보**
- 금형 ID (자동)
- 금형명 (자동)
- 요청자 (입력)
- 위치 (GPS 자동)

**2. 문제 정보**
- 긴급도 선택
  - 긴급: 생산 중단
  - 높음: 24시간 이내 처리 필요
  - 보통: 3일 이내 처리
  - 낮음: 계획 수리
- 문제 유형 선택
  - 기계적 결함
  - 전기적 문제
  - 냉각 시스템
  - 이젝터 문제
  - 표면 손상
  - 치수 불량
  - 기타
- 상세 설명 (텍스트)

**3. 사진 첨부**
- 문제 부위 사진 (필수)
- 추가 사진 (선택)

**4. 제출**
- 필수 항목 확인
- 관리자에게 즉시 알림 발송
- 요청번호 발급

**관리 기준:**
- 긴급 요청 시 관리자 즉시 알림
- 사진 최소 1장 이상 필수
- GPS 위치 자동 기록

### 4.5 정기점검

**페이지 경로:** `/worker/mold/:moldId/periodic-check`

#### 점검 프로세스

**점검 주기 확인**
- 현재 타수 확인
- 점검 레벨 확인 (1차/2차/3차)
- 점검 항목 자동 로드

**점검 항목 체크**

**1차 정기점검 (100,000타)**
- 금형 외관
- 냉각라인
- 게이트 확인

**2차 정기점검 (500,000타)**
- 캐비티/코어
- 마모 점검
- 슬라이드
- 에젝터

**3차 정기점검 (1,000,000타)**
- 전체 분해 점검
- 주요 부품 교체 판단

**점검 결과 입력**
- 항목별 상태 (양호/정비요/수리요)
- 측정값 입력
- 사진 촬영
- 수리내역 기록

**관리 기준:**
- 정기점검은 협력사 또는 전문 인력만 수행
- 모든 항목 점검 완료 필수
- 수리요 항목 발견 시 즉시 수리 계획 수립
