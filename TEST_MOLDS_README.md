## 테스트 데이터 생성 방법

### 방법 1: 서버 시작 시 자동 생성
서버의 `.env` 파일에서 다음 설정을 활성화:
```env
CREATE_SAMPLE_DATA=true
```

### 방법 2: 스크립트 직접 실행
```bash
cd server
npm run seed:test
```

## 참고사항

- 모든 테스트 금형은 `createTestMoldsWithQR()` 함수를 통해 생성됩니다.
- 테스트 금형은 실제 샘플 데이터와 별도로 관리됩니다.
- QR 코드는 `M-2024-XXX` 형식을 사용합니다.
